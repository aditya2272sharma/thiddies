
Install dependencies:

```
$ cd myapp
$ npm install
```

Install NodeMon & get started

```
$ nodemon ./bin/www
```

On MacOS or Linux, run the app with this command:

```
$ DEBUG=.:* npm start
```

On Windows, use this command:

```
> set DEBUG=.:* & npm start
```

Then load `http://localhost:3000/` in your browser to access the app.
